## ğŸ“š Study Topics

### Design and Plan Cloud Solutions
- Analyze business requirements and translate to technical requirements
- Design solutions that meet technical requirements
- Create migration plans

### Manage and Provision Infrastructure
- Network topologies (VPC, hybrid connectivity, shared VPC)
- Storage systems (Cloud Storage, Cloud SQL, Spanner, BigQuery, Firestore)
- Compute resources (Compute Engine, GKE, Cloud Run, App Engine)

### Design for Security and Compliance
- IAM, Cloud Identity, and access control
- Network security (firewall rules, Cloud Armor, VPC Service Controls)
- Data security (encryption, DLP, key management)
- Compliance requirements

### Analyze and Optimize Solutions
- Performance optimization
- Cost optimization
- Capacity planning

### Manage Implementation
- Infrastructure as Code (Terraform, Deployment Manager)
- CI/CD pipelines (Cloud Build, Artifact Registry)
- Monitoring and logging (Cloud Monitoring, Cloud Logging)

### Ensure Reliability
- High availability and disaster recovery
- Backup strategies
- Incident response
