# Contributing to Free IT Certification Resources

First off, thank you for considering contributing! This project thrives because of community input.

## How to Contribute

### Adding a Resource

1. **Fork** this repository
2. **Add** your resource to the appropriate certification file
3. **Follow** the existing format
4. **Submit** a Pull Request

### Adding a New Provider

1. Create a new folder with the provider name (lowercase, hyphens for spaces)
2. Create a `README.md` with provider overview and certification list
3. Create individual `.md` files for each certification
4. Update the main `README.md` to include the new provider

### Adding a New Certification

1. Create a new `.md` file in the appropriate provider folder
2. Use the existing certification files as a template
3. Include: Exam Overview, Official Resources, Free Practice, Video Courses, Study Topics, Hands-on Practice, Communities, Study Tips, Readiness Checklist
4. Update the provider's `README.md` to link to the new certification

### Reporting Issues

- **Broken link?** Open an issue with the broken URL
- **Outdated resource?** Let us know what changed
- **Missing certification?** Suggest it in an issue

## Code of Conduct

- Be respectful and constructive
- No piracy or braindump promotion
- Help others succeed in their certification journey
- Keep discussions focused on certifications and learning
